# Cookie Monster — Chrome Web Store Listing

## Name
Cookie Monster

## Short description (132 chars max)
Automatically removes ad trackers. Keeps your logins, carts, and subscriptions. Configures to how you browse.

## Category
Privacy & Security

## Single purpose statement
Scans and removes third-party tracking cookies from every website you visit.

## Permission justifications
- cookies: Read, classify, and delete cookies in the browser
- webRequest: Track which domains a webpage contacts to find associated cookies
- notifications: Notify users when trackers are automatically deleted
- tabs: Identify which site the user is currently viewing
- storage: Save user settings and deletion history locally
- host_permissions: Monitor cookies across all websites the user visits
